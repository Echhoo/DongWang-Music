# 音乐相关接口

1. `/music/star`
    > 用户收藏歌曲时使用的api

    使用 POST 方法
    |参数|说明|数据类型|
    |:---:|:---:|:---:|
    |uid|用户id|int|
    |mid|歌曲id|int|
    返回
    `NONE`


2. `/music/comment`
    > 用户评论时使用的api

    使用 POST 接口
    |参数|说明|数据类型|
    |:---:|:---:|:---:|
    |uid|歌手id|int|
    |mid|歌曲id|int|
    |text|评论内容|string|
    返回
    `NONE`

3. `/music/tags`
    > 获取所有标签的接口

    使用 GET 方法
    `NONE`  

4. `/music/get`
   > 获取一首歌曲的详细信息的接口

   使用 POST 方法

    |参数|说明|数据类型|
    |:---:|:---:|:---:|
    |id|歌曲id|int|
