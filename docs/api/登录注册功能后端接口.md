## 登录注册功能后端接口

|             url              |               说明                |
| :--------------------------: | :-------------------------------: |
|      /user/login    get      |         跳转到login.html          |
|     /user/login    post      | 判断用户能否登录，若成功返回token |
| /user/sendVerifyCode    post |            发送验证码             |
|    /user/register    post    |             用户注册              |
| /user/authentication    post | 判断用户是否登录，需传入token参数 |
