# 排行榜相关接口
1. `/rank/all`
    > 查询全部排行榜时使用的接口

    使用 GET 方法
    |参数|说明|数据类型|
    |:--:|:--:|:--:|
    |limit|排行榜的前几个数据|int|
    返回
    |参数|说明|数据类型|
    |:--:|:--:|:--:|
    |data|歌曲的信息|list\<songinfo\>|

2. `/rank/tag`
    > 查询某一标签歌曲的排行榜

    使用 GET 方法
    |参数|说明|数据类型|
    |:--:|:--:|:--:|
    |limit|排行榜的前几个数据|int|
    |tag|标签的名称|string|
    返回
    |参数|说明|数据类型|
    |:--:|:--:|:--:|
    |data|歌曲的信息|list\<songinfo\>|
